# AI Spec-Driven Dev Guide

A guide for specification-driven development using AI tooling.

## Overview

This repository demonstrates how to build applications using a spec-first workflow. It includes a Voice Notes reference app built with React, TypeScript, and Vite.

## Getting Started

1. Clone the repository
2. Install dependencies and run the app:

```bash
cd voice-notes-app
npm install
npm run dev
```

## Project Structure

- `voice-notes-app/` — React + TypeScript reference application
- `openspec/specs/` — Feature specifications (BDD format)
- `front-end-design/` — UI/UX design artifacts

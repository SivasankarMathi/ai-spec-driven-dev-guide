---
name: html-design
description: Convert UI design images to semantic HTML/CSS optimized for React conversion
---

Analyze the provided UI design image and convert it to clean, semantic HTML and CSS code.

Requirements:
1. **HTML Structure**:
   - Use semantic HTML5 elements (header, nav, main, section, article, footer, etc.)
   - Use meaningful class names that describe the component's purpose
   - Keep the structure flat and component-friendly for easy React conversion
   - Add data attributes or IDs where interactive elements are present

2. **CSS Styling**:
   - Write modern, clean CSS (use Flexbox/Grid for layouts)
   - Use CSS custom properties (variables) for colors, spacing, and typography
   - Make it responsive with mobile-first approach
   - Include hover states and transitions for interactive elements
   - Use BEM or similar naming convention for clarity
3. **React-Ready Considerations**:
   - Use 'className' naming convention in comments
   - Structure sections as self-contained blocks
   - Note which elements should be interactive/clickable
   - Identify reusable components in comments

4. **Analysis**:
   - First describe the layout structure and key components
   - Identify colors, fonts, spacing patterns
   - Note any interactive elements or states visible

Please provide:
- Complete HTML markup
- Complete CSS stylesheet
- Brief component breakdown for React conversion
- List of any custom fonts or icons needed
